## Frameworks

- Tailwind CSS
- Alpine.js
- Laravel
- Livewire


## Setup

```
git clone https://github.com/r3850355/i18n-teamwoker.git
cd i18n-teamwoker
composer install
npm install
npm run build
cp .env.example .env
```

## Dev Runup

```
npm run dev
php artisan serve
```

Project will running on [http://localhost:8000/](http://localhost:8000/)

## Testing

```
cp .env.example .env.testing
php artisan test
```
